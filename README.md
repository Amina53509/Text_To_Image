# 🚀 AI Image Generator (Text-to-Image)

This project is a simple **AI-powered image generator** built using Stable Diffusion and Gradio.  
It converts text prompts into high-quality images using deep learning.

## ✨ Features

- 🧠 Text to Image generation
- ⚡ Fast Mode & HD Mode
- 🚫 Negative prompt support (remove blur, distortion, etc.)
- 📥 Download generated images
- 🌐 Web UI using Gradio
- 🔥 Runs on GPU (if available)

## 🛠️ Tech Stack

- Python
- PyTorch
- Diffusers (Hugging Face)
- Stable Diffusion v1.5
- Gradio
